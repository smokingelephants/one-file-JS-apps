# Mirror Ricochet Arena

A single-file HTML/JavaScript top-down arena game where players can only damage opponents through **reflected shots**. Turrets auto-scan for the best ricochet route, mirrors move along hidden fixed paths, and **more reflections mean more damage**.

## Features

* Single HTML file, no build step
* Autoaim based on reflection routes
* Direct hits do **not** count
* Multi-hop ricochets deal higher damage
* Moving mirrors on fixed zig-zag “road” paths
* Last-survivor arena gameplay

## Controls

* **Click / Tap / Space**: fire
* **P**: pause
* **R**: restart

## Run

Open the HTML file in a browser.

## Idea

This is less a traditional shooter and more a **ricochet quick-draw duel**: wait, read the mirror geometry, and fire when a strong reflected path
appears.
